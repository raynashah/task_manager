# Task Manager App

##Overview
- List view
  - To do checklist
  - Categories: work, home, exercise, etc.
- Calendar view
  - Outlook/Gcal
- Due date, reminder alerts

##Objects/Entities
- Task
  - Description (string)
  - Due date (date)
  - Reminder alert (date)
  - Priority level (number)
  - Category
  - Is complete (boolean)
  - User
- Category
  - Name/title (string)
  - color (string)
- User //save for later
  - Username (string)
  - Password (string)
